# BookShare
A web-based platform for users to share, swap or give away books within their local community

## What it does
A mobile-first React application that guides users through listing and discovering books.  

Users can search for books using an open API, automatically populating details such as title, author, and cover image. Completed listings are stored in a database, allowing other users to search and browse available books.

## Tech stack
React, JSX, REST API integration

## Features
- Add a book listing  
- Search and browse available books
